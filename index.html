Git Neo-Guide

Complete cheat sheet with parameters & workflow visualization.

Syntax
Param
Example
Setup & Config
Initialize Repository
Starts a new Git repository in the current directory.
git init
EX git init
Clone Repository
Downloads a project and its entire version history.
git clone <url>
EX git clone https://github.com/user/repo.git
Configure User
Sets the name/email attached to your commits.
git config --global user.name "Name"
EX git config --global user.name "Jane Doe"
Branch Management
List Branches
Shows local and remote branches.
git branch -a
EX git branch -a
Safe Delete
Deletes a branch only if it has been merged.
git branch -d <name>
Ex git branch -d feature/login
Rename Current
Changes the name of the current active branch.
git branch -m <new>
Ex git branch -m feature/new-login
Remote Sync
Fetch (Clean Up)
Downloads changes and removes deleted remote branches.
git fetch --all --prune
EX git fetch --all --prune
Pull with Rebase
Fetches changes and stacks your work on top (cleaner).
git pull --rebase origin <branch>
EX git pull --rebase origin main
First Push
Uploads commits and links local branch to remote.
git push -u origin <branch>
Ex git push -u origin main
Commit & Amend
Standard Commit
Saves staged changes to history with a message.
git commit -m "Msg"
Ex git commit -m "Fix login bug"
Fix Last Commit Magic
Adds staged files to the previous commit without changing msg.
git commit --amend --no-edit
Use git commit --amend --no-edit
Switch & Reset
Create & Switch
Creates a new branch and switches to it immediately.
git switch -c <branch>
Ex git switch -c feature/dark-mode
Discard Changes
Reverts a file in the working directory to the last commit.
git restore <file>
Ex git restore style.css
Reset (Soft)
Undoes the last commit but keeps changes staged.
git reset --soft HEAD~1
Ex git reset --soft HEAD~1
Reset (Hard) Danger
Undoes last commit and DESTROYS all changes.
git reset --hard HEAD~1
Ex git reset --hard HEAD~1
Logs & Visuals
Visual Graph Log
Displays history as a graph of branches and commits.
git log --oneline --graph --all
Ex git log --oneline --graph --all
File Differences
Lists only the names of files that have changed.
git diff --name-only
Ex git diff --name-only
Stashing
Stash with Name
Temporarily saves changes with a custom message.
git stash push -m "work"
Ex git stash push -m "wip-header"
Apply & Drop
Restores the most recent stash and deletes it.
git stash pop
Ex git stash pop
